

hướng dẫn sử dụng:
chọn edit->create tile => chọn hình có tiel để tạo tile (ví dụ stage1.png)
điền width height theo pixel của mỗi tile rồi ok

nếu muốn edit bằng tay
	chọn edit->new map, chọn số dòng số cột => ok
	giờ muốn edit map thì bấm chọn từng tile rồi lắp vào khung lưới

nếu edit tự động (clone từ map có sẵn)
	chọn clone, đặt đường dẫn đến file map có sẵn ví dụ Map1.png (nên chọn file này, vì file ban đầu download lệch 2 pixel, nên không tạo map được)

chọn save.
bây giờ có ma trận index để load tile sử dụng vào game. (ví dụ trong game có file map_index.txt trong thư mục resource/map của project game)
